# VAULT

Нативный MVP для ручной организации фото и видео по локальным папкам. Требуется Xcode 15+ и iOS 17+.

## Запуск

1. Откройте `VAULT.xcodeproj` в Xcode.
2. Выберите target **VAULT** и при запуске на физическом iPhone назначьте свою Team в Signing & Capabilities.
3. Выберите симулятор iPhone с iOS 17 или физическое устройство и нажмите **Run**.
4. Для тестов используйте **Product → Test** (`⌘U`).

PhotosPicker использует системный интерфейс выбора и не требует полного доступа приложения к медиатеке. Импортированные материалы копируются в `Documents/VAULT/Media`, а уменьшенные превью — в `Documents/VAULT/Thumbnails`.

## Структура

- `App` — точка входа и SwiftData container;
- `Models` — `VaultFolder` и `VaultMediaItem`;
- `Services` — импорт, превью, локальные файлы и операции над данными;
- `ViewModels` — состояние долгих операций;
- `Views` — экраны и sheets;
- `Components` — переиспользуемые визуальные компоненты;
- `Utilities` — тема, палитра и preview fixtures;
- `Resources` — App Icon и векторные ассеты логотипа;
- `VAULTTests`, `VAULTUITests` — unit- и UI smoke tests.

## Ограничения MVP

- материалы копируются в sandbox приложения и занимают дополнительное место;
- порядок отображения сохраняется, но ручная drag-and-drop сортировка пока не добавлена;
- оригинальное имя PhotosPicker не предоставляет во всех сценариях, поэтому поле может быть пустым;
- нет синхронизации, резервного копирования и восстановления удалённых данных.
