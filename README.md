# VAULT

Нативное приложение для превращения скриншотов, фото и видео в приватную визуальную базу знаний. Требуется Xcode 15+ и iOS 17+.

## Возможности версии 1.1

- системная папка «Входящие», избранное и архив;
- шаблоны коллекций: покупки, фильмы, дизайн, игры, рецепты и путешествия;
- карточки материалов с заметкой, тегами, источником, статусом, рейтингом и ценой;
- поиск по заметкам, тегам и тексту внутри изображений;
- локальное OCR через Apple Vision и проверка дубликатов по SHA-256;
- сетка, список и доска по статусам;
- ручная drag-and-drop сортировка;
- защита Face ID или код-паролем устройства;
- экспорт VAULT Pack: JSON-манифест вместе с оригинальными файлами через системное меню «Поделиться».

## Запуск

1. Откройте `VAULT.xcodeproj` в Xcode.
2. Выберите target **VAULT** и при запуске на физическом iPhone назначьте свою Team в Signing & Capabilities.
3. Выберите симулятор iPhone с iOS 17 или физическое устройство и нажмите **Run**.
4. Для тестов используйте **Product → Test** (`⌘U`).

PhotosPicker использует системный интерфейс выбора и не требует полного доступа приложения к медиатеке. Импортированные материалы копируются в `Documents/VAULT/Media`, а уменьшенные превью — в `Documents/VAULT/Thumbnails`.

## Структура

- `App` — точка входа и SwiftData container;
- `Models` — `VaultFolder` и `VaultMediaItem`;
- `Services` — импорт, OCR, проверка дубликатов, превью, локальные файлы и операции над данными;
- `ViewModels` — состояние долгих операций;
- `Views` — экраны и sheets;
- `Components` — переиспользуемые визуальные компоненты;
- `Utilities` — тема, палитра и preview fixtures;
- `Resources` — App Icon и векторные ассеты логотипа;
- `VAULTTests`, `VAULTUITests` — unit- и UI smoke tests.

## Ограничения MVP

- материалы копируются в sandbox приложения и занимают дополнительное место;
- оригинальное имя PhotosPicker не предоставляет во всех сценариях, поэтому поле может быть пустым;
- экспорт VAULT Pack пока создаёт манифест и набор файлов; обратный импорт пакета будет следующим шагом;
- Share Extension и совместные веб-подборки требуют отдельной настройки App Group/backend и пока не включены;
- нет облачной синхронизации и восстановления удалённых данных.
