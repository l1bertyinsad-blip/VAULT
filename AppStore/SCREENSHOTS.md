# App Store screenshot package

Run the GitHub Actions workflow **App Store screenshots**. It launches the real app in an iPhone 17 Pro Max simulator, seeds local demo data only for the UI test, captures five real screens, and builds 1320×2868 PNG marketing cards.

The downloaded artifact `VAULT-App-Store-Screenshots-RU` contains:

1. `01-home.png` — “Превратите скроллинг в пользу”
2. `02-import.png` — “Сохраняйте откуда угодно”
3. `03-folders.png` — “Порядок без лишней рутины”
4. `04-favorites.png` — “Возвращайтесь к тому, что важно”
5. `05-profile.png` — “Лично. Локально. Под защитой.”

Upload them to the 6.9-inch iPhone screenshot slot in this order. They show the real interface; the added headline/background is marketing framing around the captured app screen.
